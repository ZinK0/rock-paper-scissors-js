Rock Paper Scissor Terminal Game

How to Play Rock-Paper-Scissors

Welcome to the Rock-Paper-Scissors game! This classic game is played directly in your browser's terminal. Follow the steps below to start playing:

Step 1: Open the Browser Console

  1. Open your web browser (e.g., Chrome, Firefox).
  2. Right-click anywhere on the webpage and select "Inspect" or press Ctrl+Shift+I (Windows/Linux) or Cmd+Option+I (Mac).
  3. Navigate to the "Console" tab in the Developer Tools.

Step 2: Start the Game

Once the console is open, you can begin playing by typing the following command and pressing Enter:

playGame()

Step 3: Make Your Move

The game will prompt you to choose your move. Type one of the following options and press Enter:

  • rock
  • paper
  • scissor

For example, to choose "rock," type:

"rock"

Step 4: See the Results

The game will randomly select its move and display the result in the console, letting you know if you won, lost, or tied the round.

Step 5: Play Again or Exit

  • To play another round, simply type playGame(); again and repeat the steps.
  • If you're done playing, you can close the browser console or navigate away from the page.
